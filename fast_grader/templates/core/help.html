Where do comments and grades go?

Fast Grader is currently in a pre-release stage of development. When officially released, we will have the feature to sync grades and comments to google classroom. However, that feature is not currently implemented, because the purpose of this pre-release version of the site is simply to give teachers the opportunity to test out the grading tool itself.

How does the comment bank work?

The comment bank is a bit tricky to use at first, but once you get the hang of it, it becomes blazing fast and it's a very convenient way to input comments. The comment bank uses a concept of "registers." Think of a register like a bucket that comments can be saved into and later recalled from. Registers have single-character names, which ties into how they are accessed by keyboard shortcuts.

A lowercase b is the keyboard shortcut for accessing the comment bank, then type any other character to reference a specific register of the comment bank. If no comment is already saved to that register, a window will pop up allowing you to compose one. This action will also apply the comment you just wrote for the current student. Now, you can use the same keyboard combination, b<register-name>, to apply that comment to a future student.

You can imagine that you might start a grading session by populating the comment bank with a few comments, then you can very quickly move between scholars, grade and apply comments without evet taking your hands off the keyboard.

Also, if you need to revise a comment in the comment bank, you can do that by using the uppercase "B", and then the register name just like before. This will cause the edit dialog box to come up, even if a comment is already in the comment bank. This action will update the comment bank and also apply the new comment to the current student, but will not retroactively update comments where they were used before.

Note that in the current comment bank, you cannot save comments across grading sessions, but this feature is on the feature roadmap for this project!

If you have any feedback on the comment bank, please let me know by opening a GitHub issue or sending me an email (see below).

What is the feature roadmap?

This website is currently in pre-release. There are many more features that we plan to implement ASAP!

Before Initial Release

  • Sync grades with google classroom
  • Sync comments with google classroom

After Initial Release

  • Improve differences view in grading tool to be more concise and intuitive
  • Optimize performance and page load time, especially when starting a grading session
  • Comment bank persistence: ability to save the comment bank across grading sessions
  • Widgets to select assignments to grades, instead of the current dropdown list form
  • Queue up multiple assignments to grade at the same time
  • Save comment bank across grading sessions
